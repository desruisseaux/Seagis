Browse JavaDoc

  • Browse the experimental package. This package contains a fair amount of classes (not all OpenGIS-related), requires JDK 1.4 and the generic compiler.

  • Browse the seagis package. This package contains only selected classes from the experimental package, scaled down to JDK 1.3 without generic compiler. This is the package you should use if you are looking for a stable API. This is also the package discussed in the Discussion list.

  • Browse OpenGIS specification. This package contains a copy of OpenGIS Javadoc. Note: API published there is slightly different from the one currently available on OpenGIS web site, in that most methods are declared to throw RemoteException. This detail seem to have been forgotten in the original OpenGIS's interfaces, but we don't know if and when it will be fixed.
