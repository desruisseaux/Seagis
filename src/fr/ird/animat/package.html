Représentation d'un animal virtuel. Ce paquet ne contient que des interfaces définissant une infrastructure. Le paquet fr.ird.animat.server fournira une base pour une implémentation de ces interfaces. Le paquet fr.ird.animat.server.sql fournira une implémentation plus élaborée qui puisera les données dans la base de données d'images et de pêches (paquet fr.ird.database et sous-paquets).
