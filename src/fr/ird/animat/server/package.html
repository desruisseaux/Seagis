Implémentation de base des interfaces définies dans fr.ird.animat. Cette implémentation sert de fondation au paquet fr.ird.animat.server.tuna, qui complétera l'implémentation avec un model des déplacements de thons dans le cadre du projet SEAS. En général, un modèle devrait redéfinir au moins les méthodes suivantes:
  • {@link fr.ird.animat.server.Environment#getParameters}
  • {@link fr.ird.animat.server.Environment#getCoverage(fr.ird.animat.server.Parameter)}
  • {@link fr.ird.animat.server.Animal#move}
