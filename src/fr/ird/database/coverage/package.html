Provides interfaces for accessing {@linkplain org.geotools.gc.GridCoverage grid coverages} in a database. The {@link fr.ird.database.coverage.CoverageDataBase} is the starting point, usually a connection to a database through JDBC. This object can constructs different views of tables, especially {@link fr.ird.database.coverage.SeriesTable} and {@link fr.ird.database.coverage.CoverageTable}. Those views can in turn constructs entries as a result of queries, especially {@link fr.ird.database.coverage.CoverageEntry}.
