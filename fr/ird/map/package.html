Base classes for map displays. {@link fr.ird.map.MapPanel} is the main class for this package. Since {@link fr.ird.map.MapPanel} extends {@link org.geotools.gui.swing.ZoomPane}, it support many navigation controls (keyboard, contextual menu, mouse wheel and mouse selection for zooming, translating, rotating, etc.). Layers can be added by invoking {@link fr.ird.map.MapPanel#addLayer}. Some pre-defined layers appears in the fr.ird.map.layer package.
