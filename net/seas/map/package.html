Base classes for displaying map components. {@link net.seas.map.MapPanel} is the main class for this package. Since {@link net.seas.map.MapPanel} extends {@link net.seas.awt.ZoomPane}, it support many navigation controls (keyboard, contextual menu, mouse wheel and mouse selection for zooming, translating, rotating, etc.). Layers can be added by invoking {@link net.seas.map.MapPanel#addLayer}. Some pre-defined layers appears in the net.seas.map.layer package.
