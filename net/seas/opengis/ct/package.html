Coordinate Transformations implementation. Transformations (for example map projections) are performed by {@link net.seas.opengis.ct.MathTransform}. Instances of {@link net.seas.opengis.ct.MathTransform} are created by {@link net.seas.opengis.ct.MathTransformFactory}. For example, a mercator cylindrical projection can be created by the following code:

{@link net.seas.opengis.ct.MathTransform} projection = {@link net.seas.opengis.ct.MathTransformFactory#getDefault()}.createParameterizedTransform("Mercator_1SP", null, null);

The "Mercator_1SP" string is the classification name for the mercator cylindrical projection. This package support the following classification names:

Lambert_Conformal_Conic_2SP 9802
Mercator_1SP 9804
