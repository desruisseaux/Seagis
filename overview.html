An experimental library for map visualization. This library do not follows any standard. However, if an OpenGIS's standard is made available for map visualization, we will change the API according and move it to the net.seagis package. If you are looking for an implementation of OpenGIS's standards, see the SEAGIS library instead.

 

JAR files and Dependencies

This experimental library requires the SEAGIS library (JAR files seagis-css.jar and seagis-gcs.jar). Experimental packages are grouped into two JAR files:

  • Package net.seas and subpackages contains some generic tools. They don't have any dependencies to the fr.ird package. Those packages are grouped into the seas-net.jar file.
     
  • Package fr.ird and subpackages contains an experimental application using net.seas packages. They can been seen as a demo application. Those packages are grouped into the seas-ird.jar file. Since net.seas has no dependencies to fr.ird, the whole fr directory and the seas-ird.jar file can be safely deleted without any impact on net.seas packages.
